このプログラムはDjangoによるアプリケーションとなっています。
また、画像の読み込みにopencv、畳み込み処理にnumpyをしようしているのでこれらがインストールされている必要があります。
Python3.6.0以上
Django
python-opencv
numpy

*----------
下記コマンドでwebサーバを起動します。コマンドライン上のurlにアクセスしてください
python manage.py runserver
*----------

*----------
初めのセッティングに以下のコマンドを入力して頂く必要があるかもしれません。
python manage.py makemigrations
python manage.py migrate
*----------



webアプリの操作方法
・カーネルサイズ(3か5)の選択
・フィルタ種類の選択(選択した場合は下記の入力ではなく指定したフィルタで処理を行います)
・カーネルの入力(3の場合は中心の３＊３マスが対象です)
・画像ファイルの選択
・送信する
送信が行われると選択画像とカーネルを元に畳み込みが行われます。
左画像が入力画像、右画像が出力画像として表示されます。
